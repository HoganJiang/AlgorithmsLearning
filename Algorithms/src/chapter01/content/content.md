#### 7. 异或运算  
> - 异或：无进位相加  
> - N ^ 0 = N;  
> - N ^ N = 0;  
> - 异或运算满足交换律与结合律；  
> - | -> N | 0 = N; N | 1 = 1;  
> - & -> N & 0 = 0; N & 1 = N;  
> - ^ -> N ^ 0 = N; N ^ 1 = ~N; N ^ N = 0;

#### 9. 异或运算的小技巧  
>- 求最右侧的1： N & ((~N) + 1)  
>- 消除最右侧的1：N & (N - 1)
